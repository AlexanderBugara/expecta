# Expecta

An Objective-C/Cocoa Matcher Library

## STATUS

Under development. Not quite usable at this point.

## USAGE

>`expect(x).toBeNil();` passes if `x` is `nil`
>
>`expect(x).Not.toBeNil();` passes if `x` is not `nil`
